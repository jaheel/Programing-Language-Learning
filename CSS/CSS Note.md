# CSS Note

## 1 元素

```css
font-size:12pt;#设置字体大小
font-family:"Microsoft soft";#设置字体类型

color:#ffffff;#设置字体颜色
background-color:transparent;#设置背景颜色，transparent为透明色
cursor:hand;#鼠标指向的时候变成手状
padding:1px;#离边框距离
height:30px;#高度
width:100px;#宽度

#四个边框
border-top: 2px solid #FFFFFF;
border-right: 1px solid #FFFFFF;
border-bottom: 2px solid #FFFFFF;
border-left: 1px solid #FFFFFF;

#离边框 上下 左右 距离
padding: 5px 15px;
```
